1) Why is it important to have a test set?

A test set is provided during each model evaluation to understand how the data would perform when given unseen data. The whole idea is to see if the model is performing well on data that it has not previously encountered as that would help us know how good the model is. We would also have the labels as the test set is from our own data to compare and further better the model.

2) If the normal equation always provides a solution, when would we not want to use it?

The process to calculate (XX) and then its invert form is expensive and costs O(N) where N is the number of rows/observations in X matrix. (XX) might be non-invertible, a violation of our assumption. In such cases, linearly-dependent redundant features might be removed, or regularization methods like lasso be used to lessen the number of features.

4) Explain what bias is and what variance is in regards to the bias-variance tradeoff.

Bias is the training error. It is how the model performs after being trained with some data and the residual error between the predicted values and actual values of training set. Variance is the error that a model on the test set. If the model performs too well on the train set but poorly on the test set it means it is overfitting the training set and thus we need to apply regularization methods to reduce this test error. If it is performing poorly on training and test set, the data is underfitting and we need to increase the complexity of the model.

Error is the sum of Bias, variance and the irreducible error. We need to find the sweet spot between the bias and variance where both of them are low and do not underfit/overfit

5) In a linear regression model, how might we reduce bias?

To reduce bias, increasing model complexity by tuning the hyperparameters can help. We can also form new features by transformations/ feature engineering to fit the model better. Complexity can be increased by using polynomial features, interaction terms, or adding more variables to the model.

6) In a linear regression model, how might we reduce variance?

To reduce variance, decreasing the complexity of the model by removing irrelevant features or applying regularization can aid in doing so. Using techniques like k-fold cross-validation can provide estimates of the model's performance on unseen data, helping to identify and mitigate overfitting.
